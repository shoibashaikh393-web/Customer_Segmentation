# Customer Segmentation Dashboard - Meesho Inspired Analysis

## 📌 Overview
This project focuses on segmenting customers based on their purchasing behavior using Machine Learning and visualizing business insights through an interactive Power BI dashboard.

The objective is to help businesses identify different customer groups and create targeted marketing strategies to improve customer retention and revenue.

---

## 🎯 Objectives
- Perform customer segmentation using K-Means Clustering.
- Analyze customer purchase behavior and spending patterns.
- Generate actionable business insights.
- Build an interactive Power BI dashboard.

---

## 🛠️ Technologies Used
- Python
- Pandas
- NumPy
- Scikit-learn
- K-Means Clustering
- Power BI
- DAX
- Data Visualization

---

## 📊 Dashboard Features
- Total Customers
- Total Revenue
- Average Order Value
- Average Discount
- Customer Distribution by Cluster
- Revenue by Segment
- Orders by Segment
- Customer Segmentation Scatter Plot
- Interactive Filters and Slicers

---

## 🤖 Machine Learning Workflow
1. Data Cleaning and Preprocessing
2. Feature Engineering
3. Data Standardization
4. Elbow Method for Optimal Clusters
5. K-Means Clustering
6. Cluster Analysis and Interpretation

---

## 📈 Business Insights
- Identify premium customers for loyalty programs.
- Understand discount-sensitive customers.
- Improve targeted marketing campaigns.
- Support data-driven decision making.
